## 先想清楚：这讲到底在解什么问题？

接着上一讲的场景：你有一台机械臂，要让末端的螺丝刀对准桌上的一颗螺丝。你知道螺丝在三维空间中的位置和朝向（也就是你希望末端到达的**目标位姿**）。

问题是：**每个关节应该转到什么角度，才能让末端精确到达那个位姿？**

这就是逆运动学（Inverse Kinematics, IK）——前向运动学的**反过程**。

## 1. IK 的问题定义

> 📖 **对照 PPT**：打开 lecture02.pdf **第 22 页** "Inverse Kinematics 2D"，
> 图中画了一个单关节的例子：已知 ${}^A R_B$ 和 ${}^A \mathbf{t}_B$，要求关节角 $q$。

**正向运动学**（已解决）：

$${}^{Base} E_{Tool} = f(q_1, q_2, \ldots, q_N)$$

**逆运动学**（本讲问题）：

$$\mathbf{q} = f^{-1}({}^{Base} E_{Tool})$$

给定末端位姿（$4 \times 4$ 齐次变换矩阵），反推所有关节角。

### 为什么 IK 比 FK 难得多？

| FK | IK |
|----|----|
| 就是矩阵乘法，一步到位 | 要"解方程" |
| 解是唯一的 | 可能有**多个解**（多种姿态能到同一点） |
| 一定有解 | 可能**无解**（目标超出工作空间） |
| 计算简单 | 对复杂机械臂**没有通用的解析解** |

---

## 2. 简单情况：单关节解析解

> 📖 **对照 PPT**：lecture02.pdf **第 22 页** 给了一个单关节例子。

如果只有一个旋转关节，$q$ 只出现在旋转矩阵 $R$ 中：

$${}^A R_B(q) = \begin{pmatrix} \cos q & -\sin q \\ \sin q & \cos q \end{pmatrix}$$

给定目标 $R$，可以直接读出 $q$：

$$q = \text{atan2}(\sin q, \cos q) = \text{atan2}(R_{21}, R_{11})$$

PPT 的例子：$R_{11} = 0.7071$，$R_{21} = 0.7071$ → $q = 45°$。

这种情况很简单——但真实机器人有 6 个关节，方程组会复杂得多。

## 3. 复杂情况：多关节 IK

> 📖 **对照 PPT**：打开 lecture02.pdf **第 23 页** "Inverse Kinematics 3D"，
> PPT 明确写了两个关键事实：
> A) 多关节的符号解非常难获得（"difficult to get"）
> B) 数值解（Newton 法）更通用（"more generic"）

### 3.1 解析解（符号解）

对于一些**特殊结构**的机械臂（比如所有旋转轴相交于一点），可以用三角恒等式推导出关节角的公式。

**优点**：精确、快速
**缺点**：每种机械臂结构都需要单独推导，没有通用方法

### 3.2 数值解（迭代法）

对于一般的机械臂，用**数值迭代**方法：

1. 猜一组初始关节角 $\mathbf{q}_0$
2. 用 FK 算出当前末端位姿 $E(\mathbf{q}_0)$
3. 和目标位姿比较，计算误差
4. 用 Newton 法（或类似方法）更新 $\mathbf{q}$
5. 重复直到误差足够小

**优点**：通用，适用于任何结构
**缺点**：可能收敛到局部最优、依赖初始猜测、速度相对慢

### 3.3 IK 的多解性

> PPT 第 23 页底部有一个重要提醒：
> "Note that the inverse kinematics is NOT the inverse of the forward kinematics ($E^{-1}$)"

$$\mathbf{q} = f^{-1}(E_{target}) \neq E_{target}^{-1}$$

$E^{-1}$ 是矩阵求逆（总是存在的），而 $f^{-1}$ 是函数求逆（可能有多个解或无解）——这完全不是一回事！

**直觉例子**：你伸手去够桌上的一个杯子。你可以"肘部朝上"够到，也可以"肘部朝下"够到。同一个末端位置，有两种（甚至更多种）关节角组合。

对于工业 6 轴机械臂，一个目标位姿最多可能有 **8 组**不同的关节角解！

## 4. 数值算例：2D 两关节 IK

**题目**：一个 2D 两关节机械臂，$L_1 = L_2 = 1.0$。目标末端位置为 $(1.0, 1.0)$。求关节角 $q_1, q_2$。

这个简单情况可以用几何法解析求解：

**Step 1**：末端到基座的距离
$$d = \sqrt{x^2 + y^2} = \sqrt{1.0^2 + 1.0^2} = \sqrt{2} \approx 1.414$$

**Step 2**：用余弦定理求 $q_2$
$$\cos q_2 = \frac{d^2 - L_1^2 - L_2^2}{2 L_1 L_2} = \frac{2 - 1 - 1}{2} = 0$$

$$q_2 = \pm 90°$$

两个解！对应"肘部朝上"和"肘部朝下"。

**Step 3**：求 $q_1$（以 $q_2 = 90°$ 为例）

$$q_1 = \text{atan2}(y, x) - \text{atan2}(L_2 \sin q_2, L_1 + L_2 \cos q_2)$$
$$= \text{atan2}(1, 1) - \text{atan2}(1 \cdot 1, 1 + 1 \cdot 0) = 45° - 45° = 0°$$

**验证**（FK）：$q_1 = 0°, q_2 = 90°$
- 关节 1 不旋转，连杆 1 沿 x 方向，末端到 $(1, 0)$
- 关节 2 旋转 90°，连杆 2 沿 y 方向，末端到 $(1, 1)$ ✓

```python
import numpy as np

L1, L2 = 1.0, 1.0
x_target, y_target = 1.0, 1.0

d = np.sqrt(x_target**2 + y_target**2)
cos_q2 = (d**2 - L1**2 - L2**2) / (2 * L1 * L2)
q2 = np.arccos(np.clip(cos_q2, -1, 1))  # 取正值 = "肘部朝上"

q1 = np.arctan2(y_target, x_target) - np.arctan2(L2*np.sin(q2), L1 + L2*np.cos(q2))

print(f"解 1: q1 = {np.degrees(q1):.1f}°, q2 = {np.degrees(q2):.1f}°")

# 第二组解（肘部朝下）
q2_alt = -q2
q1_alt = np.arctan2(y_target, x_target) - np.arctan2(L2*np.sin(q2_alt), L1 + L2*np.cos(q2_alt))
print(f"解 2: q1 = {np.degrees(q1_alt):.1f}°, q2 = {np.degrees(q2_alt):.1f}°")

# 用 FK 验证两组解
def fk_2d(q1, q2, L1, L2):
    x = L1*np.cos(q1) + L2*np.cos(q1 + q2)
    y = L1*np.sin(q1) + L2*np.sin(q1 + q2)
    return x, y

print(f"\n验证解 1: {fk_2d(q1, q2, L1, L2)}")
print(f"验证解 2: {fk_2d(q1_alt, q2_alt, L1, L2)}")
# 两组解都应该得到 (1.0, 1.0)
```

**输出**：
```
解 1: q1 = 0.0°, q2 = 90.0°
解 2: q1 = 90.0°, q2 = -90.0°

验证解 1: (1.0, 1.0)
验证解 2: (1.0, 1.0)
```

## 5. 本讲小结

| 概念 | 要点 |
|------|------|
| **逆运动学 (IK)** | 给定目标末端位姿，求关节角 |
| **解析解** | 特定结构可以用公式求解，精确但不通用 |
| **数值解** | 迭代求解（Newton 法），通用但可能陷入局部最优 |
| **多解性** | 同一个目标位姿可能对应多组关节角 |
| **无解** | 目标在工作空间之外时无解 |
| **IK ≠ $E^{-1}$** | IK 是函数反求（困难），不是矩阵求逆（简单） |

**学完本讲你应该能**：

2. 区分解析解和数值解的优缺点
3. 对 2D 两关节机械臂用几何法手算 IK
