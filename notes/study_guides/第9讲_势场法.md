## 先想清楚：这讲到底在解什么问题？

上一讲我们把运动规划问题转化为"在 C-space 中从 A 点到 B 点，避开障碍区域"。现在问题是：**具体怎么找这条路径？**

势场法的灵感来自物理学：想象目标点像一块磁铁，把机器人"吸"过去；障碍物像同极的磁铁，把机器人"推"开。机器人在这两种"力"的合力下移动，最终到达目标。

这是最直观的运动规划方法——不需要搜索、不需要图论，只需要算梯度然后走。

## 1. 核心思想：人工势场

> 📖 **对照 PPT**：potential.pdf **第 7–8 页** 展示了从 A 到 B 的路径规划，
> 障碍物将 C-space 分隔成自由区域和碰撞区域。
>
> Path_planning_image-based_control.pdf **第 2–3 页**（Section I-B）详细定义了势场法。

### 1.1 总势场 = 引力 + 斥力

定义一个**人工势场函数** $U(\mathbf{q})$，在 C-space 的每个点都有一个"高度"值：

$$U(\mathbf{q}) = U_{att}(\mathbf{q}) + U_{rep}(\mathbf{q})$$

| 分量 | 作用 | 特点 |
|------|------|------|
| $U_{att}$（引力势） | 把机器人拉向目标 | 在目标处 = 0，离目标越远越高 |
| $U_{rep}$（斥力势） | 把机器人推离障碍物 | 在障碍物附近很高，远离障碍物时 = 0 |

### 1.2 梯度下降 = "滚下坡"

势场就像一个地形：目标是山谷（最低点），障碍物是山峰（高点）。机器人像一个球，沿着**地形最陡的下坡方向滚动**。

数学上就是**负梯度方向**：

$$\mathbf{F}(\mathbf{q}) = -\nabla U(\mathbf{q}) = -\nabla U_{att}(\mathbf{q}) - \nabla U_{rep}(\mathbf{q})$$

路径生成的迭代公式：

$$\mathbf{q}_{k+1} = \mathbf{q}_k + \alpha \cdot \mathbf{F}(\mathbf{q}_k)$$

其中 $\alpha > 0$ 是步长。

---

## 2. 引力势场

> 📖 **对照 PPT**：Path_planning_image-based_control.pdf **第 4 页** Section III-D "Attractive Potential and Force"。

最简单的引力势是**二次函数**（抛物面）：

$$U_{att}(\mathbf{q}) = \frac{1}{2} \|\mathbf{q} - \mathbf{q}_{goal}\|^2$$

- 在目标处 $U_{att} = 0$（最低点）
- 离目标越远，势越高

对应的引力：

$$\mathbf{F}_{att}(\mathbf{q}) = -\nabla U_{att} = -(\mathbf{q} - \mathbf{q}_{goal}) = \mathbf{q}_{goal} - \mathbf{q}$$

**直觉**：引力指向目标，大小和距离成正比——离得越远，拉力越大。

> 如果只有引力、没有障碍物，路径就是从起点到终点的**直线**——这是最简单的情况。

---

## 3. 斥力势场

障碍物附近要有排斥力，远离障碍物时斥力消失。标准的斥力势定义：

$$U_{rep}(\mathbf{q}) = \begin{cases} \frac{1}{2}\eta \left(\frac{1}{d(\mathbf{q})} - \frac{1}{d_0}\right)^2 & \text{if } d(\mathbf{q}) \leq d_0 \\ 0 & \text{if } d(\mathbf{q}) > d_0 \end{cases}$$

其中：
- $d(\mathbf{q})$ = 当前构型到最近障碍物的距离
- $d_0$ = 障碍物的**影响范围**（超过这个距离就不管了）
- $\eta$ = 斥力强度系数

**特点**：
- 离障碍物越近，$1/d$ 越大，斥力越强
- 碰到障碍物时（$d \to 0$），斥力趋向无穷大
- 距离超过 $d_0$ 时，斥力完全消失

---

## 4. 完整的势场法算法

```
输入: q_start, q_goal, 障碍物列表, 步长 α, 最大迭代数
输出: 从 q_start 到 q_goal 的路径

1. q ← q_start
2. path ← [q]
3. 重复：
   a. F_att ← -(q - q_goal)                    # 引力
   b. F_rep ← 计算所有障碍物的斥力之和            # 斥力
   c. F ← F_att + F_rep                         # 合力
   d. q ← q + α · F / ||F||                     # 沿合力方向走一步
   e. path.append(q)
   f. 如果 ||q - q_goal|| < 阈值: 成功，返回 path
   g. 如果迭代次数超限: 失败
```

## 5. 数值算例：2D 势场法

**题目**：在 2D C-space 中，起点 $(0, 0)$，目标 $(8, 8)$，有一个圆形障碍物在 $(4, 4)$，半径 $1.5$。用势场法规划路径。

```python
import numpy as np

q_start = np.array([0.0, 0.0])
q_goal = np.array([8.0, 8.0])
obs_center = np.array([4.0, 4.0])
obs_radius = 1.5
d0 = 3.0     # 斥力影响范围
eta = 50.0   # 斥力强度
alpha = 0.05 # 步长

def attractive_force(q):
    return q_goal - q

def repulsive_force(q):
    diff = q - obs_center
    d = np.linalg.norm(diff) - obs_radius  # 到障碍物表面的距离
    if d <= 0:
        d = 0.001  # 防止除以零
    if d > d0:
        return np.zeros(2)
    magnitude = eta * (1/d - 1/d0) * (1/d**2)
    direction = diff / np.linalg.norm(diff)  # 远离障碍物的方向
    return magnitude * direction

# 迭代
path = [q_start.copy()]
q = q_start.copy()

for i in range(2000):
    F = attractive_force(q) + repulsive_force(q)
    q = q + alpha * F / (np.linalg.norm(F) + 1e-10)
    path.append(q.copy())
    if np.linalg.norm(q - q_goal) < 0.2:
        print(f"到达目标！迭代 {i+1} 次")
        break

path = np.array(path)
print(f"路径点数: {len(path)}")
print(f"最终位置: ({path[-1][0]:.2f}, {path[-1][1]:.2f})")
print(f"离目标距离: {np.linalg.norm(path[-1] - q_goal):.3f}")
```

**预期结果**：路径从 $(0,0)$ 出发，大致沿对角线走向 $(8,8)$，但在接近 $(4,4)$ 的障碍物时会被推开，绕一个弧线后继续趋向目标。

## 6. 局部极小值问题

### 6.1 什么是局部极小值？

当引力和斥力恰好**大小相等、方向相反**时：

$$\mathbf{F}_{att} + \mathbf{F}_{rep} = \mathbf{0}$$

机器人停下来了——但它**不在目标处**！这就是**局部极小值**（Local Minimum）。

**直觉**：想象一个球滚下山坡，但半路上有一个小坑，球掉进坑里就出不来了——这个坑不是山谷（目标），但球以为到了。

### 6.2 什么时候容易出现？

- 障碍物正好在起点和目标的连线上
- U 形死胡同（斥力从三面包围）
- 多个障碍物形成"陷阱"

### 6.3 解决方案

| 方法 | 思路 |
|------|------|
| **随机扰动** | 检测到卡住后，加一个随机力跳出局部极小值 |
| **导航函数** | 特殊设计的势函数，保证没有局部极小值（但构造困难） |
| **采样规划（PRM/RRT）** | 完全不同的思路——不用势场，用随机采样（下一讲） |

> PPT 和论文都提到：虽然局部极小值理论上存在，但在**六维空间**中（6 轴机械臂），所有 6 个分量同时为零的概率很低，实际中势场法通常能工作。

## 7. 势场法用于机械臂

> 📖 **对照 PPT**：potential.pdf **第 7–8 页** 和期中复习 review_midterm.pdf **第 3 页** 提到：
> "Robot arms use implicit potential in the workspace and the Jacobian to relate workspace potential to configuration space potential"

对于机械臂，势场通常定义在**工作空间**中（三维空间中末端离目标多远、离障碍物多近），然后用 **Jacobian** 转换到关节空间：

$$\tau = J^T(\mathbf{q}) \cdot \mathbf{F}_{workspace}$$

其中 $\tau$ 是关节空间中的"力"（实际上是关节速度的方向），$J^T$ 把工作空间的力映射回关节空间。

> 这就是第 5 讲学的 Jacobian 的另一个应用！

## 8. 本讲小结

| 概念 | 要点 |
|------|------|
| **引力势 $U_{att}$** | 二次函数，在目标处最低，产生指向目标的力 |
| **斥力势 $U_{rep}$** | 在障碍物附近很高，远处为零，产生远离障碍物的力 |
| **势场法** | 合力 = 引力 + 斥力，沿合力方向迭代移动 |
| **局部极小值** | 合力为零但不在目标处，机器人卡住 |
| **Jacobian 转置** | 把工作空间的势场力转换到关节空间 |
