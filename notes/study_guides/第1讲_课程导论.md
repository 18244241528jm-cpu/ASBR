## 先想清楚：这讲到底在解什么问题？

想象你买了一台扫地机器人。你把它放在客厅，按下启动键，然后它需要：

1. **自己搞清楚房间长什么样**（哪里有沙发、哪里有墙），同时**知道自己在房间里的哪个位置**——这就是 **SLAM**（同时定位与建图）。
2. **规划一条路径**，从当前位置移动到还没扫过的角落，同时**不撞到任何家具**——这就是**运动规划**（Motion Planning）。

这门课（Algorithms for Sensor-Based Robotics）就是围绕这两个核心问题展开的。

## 2. 核心问题一：运动规划（Motion Planning）

> 📖 **对照 PPT**：打开 lecture01.pdf **第 3–5 页** "What is Motion Planning?" 和 "Basic Path Planning"，
> 第 5 页有一个 2D 示意图：一个机器人（彩色方块）需要从起点到终点，中间有灰色障碍物，需要绕行。

### 问题定义

**运动规划**的核心问题是：

> 给定一个机器人、一个已知的环境、一个起点和一个终点，
> 计算一条从起点到终点的**连续路径**，使机器人**不与任何障碍物碰撞**。

用更正式的话说，输入是：
1. 环境的几何信息（障碍物在哪）
2. 机器人的几何和运动学信息（机器人长什么样、怎么动）
3. 起始位姿和目标位姿

输出是：一条**无碰撞路径**（collision-free path）。

### 但现实没那么简单

> 📖 **对照 PPT**：打开 lecture01.pdf **第 6 页** "What did we assume?"，
> 列出了三个被忽略的问题。

上面的问题定义做了很多**理想化假设**：

| 假设 | 现实 |
|------|------|
| 传感器完美 | 传感器有噪声、有盲区 |
| 控制完美 | 电机有误差、轮子会打滑 |
| 对环境完全了解 | 很多时候地图不完整甚至完全未知 |

这门课的后半段（滤波、SLAM）就是在放松这些假设后，用概率方法处理不确定性。

### 运动规划不只是"导航"

> 📖 **对照 PPT**：打开 lecture01.pdf **第 7–9 页**，
> 第 7 页展示了药物设计中的分子路径规划，第 8 页展示了有运动学约束的汽车/拖车规划，
> 第 9 页展示了考虑动力学的运动。

运动规划不仅限于"让小车从 A 走到 B"。它的应用远比你想象的广：

- **药物设计**：分子在蛋白质口袋中的路径（概率方法最早就来自这个领域！）
- **自动驾驶**：汽车不能侧移，有运动学约束
- **机械臂操作**：6 个关节的高维空间规划

**这门课会学到的规划方法**：
- **势场法**（Potential Fields）：用"引力"吸引机器人到目标、用"斥力"推开障碍物
- **采样规划**（PRM、RRT）：随机撒点，连成路线图，在图上搜索路径

## 3. 核心问题二：SLAM（同时定位与建图）

> 📖 **对照 PPT**：打开 lecture01.pdf **第 10 页** "SLAM"，
> 用了一个比喻：假设你在沙漠里醒来，需要找路——怎么做？

### 鸡生蛋还是蛋生鸡？

SLAM 面对的是一个经典的**鸡和蛋问题**：

- **定位**（Localization）：我在地图的什么位置？→ 但你**没有地图**。
- **建图**（Mapping）：环境长什么样？→ 但你**不知道自己在哪**。

你需要**同时**解决这两个问题——这就是 SLAM（Simultaneous Localization And Mapping）。

### 直觉理解

想象你蒙着眼睛被放进一个陌生的房间：

1. 你伸手摸到一面墙——你知道"附近有墙"，但不知道自己在房间哪个角落。
2. 你沿着墙走了几步，又摸到一个门框——现在你既更新了对房间的理解（有墙、有门），也根据走的步数大概知道了自己的位移。
3. 你继续走，不断**用新的感知更新地图**，同时**用地图修正自己的位置估计**。

这个"边走边测、边测边改"的循环就是 SLAM 的核心思想。

> PPT 第 11–16 页展示了不同场景下的 SLAM 效果图，包括室内走廊、室外街区等场景。
> 可以看到随着机器人移动，地图逐渐被构建出来。

**这门课会学到的 SLAM/定位方法**：
- **卡尔曼滤波**（Kalman Filter）：处理高斯噪声下的最优估计
- **扩展卡尔曼滤波**（EKF）：处理非线性系统
- **贝叶斯滤波**（Bayes Filter）：概率推断的通用框架
- **粒子滤波**（Particle Filter）：用一堆"粒子"近似概率分布

## 4. 这门课的整体地图

> 📖 **对照 PPT**：打开 lecture01.pdf **第 18 页** "This Course"，
> 列出了课程的两大聚焦方向和关键假设。

整门课按时间顺序分为两大部分：

```
┌─────────────────────────────────────────────────────┐
│                  ASBR 课程地图                        │
├─────────────────────┬───────────────────────────────┤
│   前半学期（期中前）    │    后半学期（期中后）           │
│                     │                               │
│  ① 刚体运动 & 运动学  │  ⑤ 卡尔曼滤波                 │
│  ② 手眼标定          │  ⑥ 扩展卡尔曼 & SLAM           │
│  ③ 构型空间 & 势场法  │  ⑦ 贝叶斯滤波                 │
│  ④ 采样规划 (PRM/RRT)│  ⑧ 粒子滤波                   │
│                     │                               │
│  关键词：几何、代数    │  关键词：概率、统计              │
├─────────────────────┴───────────────────────────────┤
│  贯穿始终的假设：                                      │
│  • 静态世界（环境不变化）                                │
│  • 不考虑动力学（只关心位置，不关心力和加速度）             │
└─────────────────────────────────────────────────────┘
```

**两个关键假设**（PPT 明确提到）：
1. **静态世界**：环境不会变化（没有移动的障碍物）
2. **忽略动力学**：只关心几何路径，不关心机器人的质量、惯性、力

这些简化让我们可以专注于算法本身。

## 6. 本讲小结

| 概念 | 要点 |
|------|------|
| **运动规划** | 给定环境和起终点，计算无碰撞路径 |
| **SLAM** | 在未知环境中同时建立地图和定位自己 |
| **前半学期** | 运动学 → 手眼标定 → 构型空间 → 路径规划（几何/代数方法） |
| **后半学期** | 卡尔曼滤波 → EKF → 贝叶斯滤波 → 粒子滤波（概率方法） |
| **关键假设** | 静态世界、忽略动力学 |
